# Excel公式急救箱 1.0.3

正式驗收版。使用流程：

1. 首頁選擇六大症狀之一。
2. 第二畫面選擇最符合需求的公式卡。
3. 彈窗查看公式、修改方式與範例，並一鍵複製。
4. 首頁可直接點「安裝到手機」；iPhone會顯示加入主畫面的操作步驟。

## Windows本機啟動

完整解壓縮後，雙擊：

- `啟動Excel公式急救箱.bat`
- 或 `START-HERE.bat`

第一次啟動會自動安裝必要套件。本機預覽會從 `43401` 到 `43419` 尋找可用埠號。

## 開發指令

```bash
npm install
npm run dev
npm run verify
npm run build
```

## PWA安裝

- iPhone／iPad：首頁點「安裝到手機」，依彈窗指示用Safari的「分享」→「加入主畫面」。
- Android／電腦：首頁點「安裝到手機」，瀏覽器支援時會直接顯示安裝視窗。

## 正式部署

可將專案連接GitHub後部署到Vercel。正式建置輸出位於 `dist/`。
